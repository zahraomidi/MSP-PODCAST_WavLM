Learning from Annotation Uncertainty

Entropy-Aware Curriculum for Speech Emotion Recognition

Speech emotion recognition commonly relies on hard consensus labels that collapse annotator disagreement into a single category. This project studies distribution-based supervision in a WavLM-Base SER system on the 9-class MSP-Podcast 2.0 benchmark, comparing hard consensus training with targets derived from primary and merged primary-secondary annotator vote distributions.

arXiv Paper PDF Code

Task

Speech emotion recognition on the 9-class MSP-Podcast 2.0 benchmark.

Supervision

Hard consensus labels are compared with primary and merged primary-secondary annotator vote distributions.

Architecture

A unified multitask architecture jointly models categorical emotion and dimensional VAD.

Model Architecture

Model architecture. WavLM-Base is used as the backbone, followed by a TC-GRU model. The model jointly predicts a discrete emotion distribution and dimensional emotion attributes including valence, activation, and dominance.

Supervision Targets

Hard

Hard consensus training collapses annotator disagreement into a single emotion category.

Primary

Primary distribution targets are derived from primary annotator vote distributions.

Merged

Merged targets combine primary and secondary annotator vote distributions.

Entropy

Entropy-aware curricula use annotation uncertainty during training.

Supervision and Annotation Uncertainty

Per-class annotation entropy. Entropy is shown under primary and merged primary-secondary vote distributions. Merged targets use 0.9P/0.1S and 0.8P/0.2S settings.

Main Findings

JSD/KLD

Distributional objectives consistently improve alignment with annotator vote distributions, yielding lower JSD and KLD.

Other

Gains under hard supervision are partly driven by assigning ambiguous samples to the residual Other category.

Entropy

Entropy-stratified evaluation shows that high-ambiguity utterances remain challenging.

Uncertainty

Distribution-based supervision provides more faithful modeling of perceptual uncertainty.

Embedding Visualization

Utterance-level embedding visualization. UMAP projections compare Hard-CBCE, M90 entropy filtering, and M90 entropy weighting.

Conclusion

These findings suggest that SER systems should move beyond hard consensus labels toward distributional targets that better reflect listener disagreement.
