Learning from Annotation Uncertainty

Entropy-Aware Curriculum for Speech Emotion Recognition

This project studies distribution-based supervision for SER using WavLM on MSP-Podcast 2.0. Instead of collapsing listener disagreement into one hard label, we train models with annotator vote distributions and analyze how uncertainty affects emotion recognition.

arXiv Paper PDF Code

Problem

Hard consensus labels hide annotator disagreement and can collapse ambiguous samples into a single category.

Idea

Use primary and merged primary-secondary vote distributions as supervision targets.

Finding

Distributional objectives reduce JSD/KLD and better reflect perceptual uncertainty.

Main Findings

↓ JSD/KLD

Distribution-based supervision improves alignment with annotator vote distributions compared with hard-label training.

Other class

Hard supervision partly benefits by assigning ambiguous utterances to the residual Other category.

Entropy

High-entropy utterances remain challenging, showing that perceptual ambiguity is not fully solved by hard categorical prediction.

Uncertainty

Distributional targets better preserve listener disagreement by spreading probability mass across structured emotion categories.
